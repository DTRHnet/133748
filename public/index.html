Help & Usage

Search Functionality

Enter an artist and song name to search for tabs on Ultimate Guitar. Examples:

  • metallica one
  • dream theater pull me under
  • plini electric sunrise

Download Functionality

Paste a direct URL to an Ultimate Guitar tab page to download it. The URL should look like:

https://tabs.ultimate-guitar.com/tab/metallica/one-guitar-pro-54157

Options

  • Guitar Pro only: Filter results to show only Guitar Pro tabs
  • JSON output: Return results in JSON format for API usage
  • Limit: Maximum number of results to return

API Endpoints

  • GET /api/search?query=metallica+one
  • POST /api/grab with {"url": "...", "filename": "..."}
  • GET /api/health for system status
